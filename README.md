# FSL_Research
研究用プログラム

## 各ファイル説明

|ファイル名|説明|
|--|--|
|selfsupervised(フォルダ)|自己教師あり学習用|
|args.py|シミュレーション時の引数一覧|
|augmentations.py|データ拡張用|
|backbones.py|バックボーンの設定用|
|classifiers.py|分類器の設定・学習用|
|dataloaders.py|評価・学習で扱うデータの処理用|
|few_shot_evaluation.py|Few-Shot Learningにおけるエピソード生成用|
|main.py|評価フローを回すメイン関数|
|utils.py|ユーティリティ関数群|
|evaluation.sh|基本引数を設定して評価を始めるスクリプト|
